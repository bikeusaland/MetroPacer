# MetroPacer — App Review Information

Paste into App Store Connect → your version → **App Review Information**.

## Notes for the reviewer
```
MetroPacer is a running-cadence metronome that layers a click over the user's
own audio. To test the core feature: open Music, Spotify, or Podcasts and press
play, then return to MetroPacer, set a cadence (use a preset such as 180 or the
slider), and press Start — the metronome beat plays on top of the background
audio without stopping it.

Background audio mode (UIBackgroundModes: audio) is used so the beat continues
while the screen is locked or the user is in another app during a run.

Optional feedback: in Options, "Flash" pulses the screen on each beat and
"Vibrate" gives a haptic tap — useful outdoors when the click is hard to hear.

The app requires no sign-in and collects no data. All settings are stored
locally on device.
```
